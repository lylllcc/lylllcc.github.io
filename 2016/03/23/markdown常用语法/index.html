2016-03-23

markdown的11中常用语法

######1.标题
用# + “文字” 表示标题，一个#表示一级标题两个代表二级最高六级

#这是一级标题

##这是二级标题

###这是三级标题

######2.注释块
可以用 > + “文字” 表示注释块

这是一个注释块
这是注释块的第二行

用> + 五个空格 + 文字表示注释块中的注释

这是一个小的注释快中的注释

这是注释块的第二行

######3.斜体
将要设置为斜体的文字两端使用1个“*”或者“_”夹起来

这是一个斜体

这也是一个斜体

######4.粗体
将要设置为斜体的文字两端使用2个“*”或者“_”夹起来

这是一个粗体

这也是一个粗体

######5.无序列表
用 “-” +“文字” 或 “+” “文字” 或“*” +“文字” 表示无序列表 注意空格 一次只能使用一种

  • 无序一
  • 无序二
  • 无序三

######6.有序列表
使用“数字”+“.” + “文字” 注意空格

  1. 列表一
  2. 列表二
  3. 列表三
